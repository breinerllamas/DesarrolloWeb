Tipos de datos guardados

Con HTML5 ya es posible guardar datos locales en el ordenador del usuario sin limitación de tiempo, sin que se borren en cada sesión, con una limitación de hasta 5 MB, sin necesitar ningún servidor, base de datos ni cookies.

Existen dos tipos de almacenamiento:

  • sessionStorage: los datos permanecen mientras el usuario tiene el navegador abierto. Al cerrar el navegador se borran automáticamente.
  • localStorage: los datos permanecen siempre, incluso si se cierra el navegador y se apaga el ordenador.

Es importante señalar que la información guardada es accesible solo desde el mismo navegador que la guardó. El localStorage de Chrome es diferente al de Firefox o al de Opera.

Las palabras mágicas

El localStorage funciona bajo JavaScript, por lo que el código debe colocarse dentro del <head>, entre estas dos etiquetas:

<script type="text/javascript">
</script>

Guardar un dato

Para guardar un valor necesitamos 3 datos:

  • a) localStorage (permanente) o sessionStorage (solo mientras el navegador está abierto).
  • b) Un nombre para identificar esa información.
  • c) La información a guardar.
localStorage.nombre = 'Javier';

Este código guarda permanentemente el dato "Javier" dentro del dato llamado nombre.

Recuperar la información

Para leer o recuperar la información necesitamos también 3 datos:

  • a) localStorage o sessionStorage según corresponda.
  • b) El nombre que le dimos al dato cuando se guardó.
  • c) Una variable donde guardar dicha información (o un alert que la muestre).
alert('La información recuperada ha sido ' + localStorage.nombre);

El alert hace aparecer una ventana flotante con el texto indicado más el valor guardado en localStorage.nombre.

Eliminar un dato concreto

localStorage.removeItem('nombre');

Elimina únicamente el dato llamado nombre del localStorage permanente.

Eliminar todos los datos

localStorage.clear();
sessionStorage.clear();

Elimina todos los datos guardados, tanto permanentes como los de la sesión actual.

Saber cuántos datos hay guardados

alert('Hay ' + localStorage.length + ' elementos guardados');

Con el atributo .length podemos conocer cuántos elementos hay guardados en el localStorage.

Ejemplo práctico paso a paso

A continuación se muestra un contador que va contando las veces que has entrado en esta página con este navegador. Recarga la página varias veces para comprobar cómo se incrementa, incluso si cierras el navegador y vuelves mañana.

Cargando...

El código JavaScript que lo hace posible, colocado dentro del <head>, es el siguiente:

<script type="text/javascript">

  if (localStorage.veces == undefined) {
    localStorage.veces = 1;
  } else {
    localStorage.veces = Number(localStorage.veces) + 1;
  }

  document.write("Visitas: " + localStorage.veces + " veces.");

</script>

Explicación línea a línea

  • Usamos localStorage para que los datos sean permanentes (no se borren al cerrar el navegador).
  • Llamamos al dato veces, por lo que accedemos a él como localStorage.veces.
  • if (localStorage.veces == undefined): si el dato no existe todavía (primera visita), lo inicializamos a 1.
  • localStorage.veces = 1;: guarda el valor 1 en la primera visita.
  • } else {: si el dato ya existe (ya hemos visitado la página antes)...
  • localStorage.veces = Number(localStorage.veces) + 1;: le suma 1 al valor actual. El Number() es necesario para que haga una suma (1+1=2) y no una concatenación de texto (1+1="11").
  • document.write(...): escribe el resultado directamente en la página web.
