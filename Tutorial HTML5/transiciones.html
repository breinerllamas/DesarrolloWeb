Transition

Las transiciones (transition) funcionan como pequeñas animaciones que siguen las mismas características que transform (cambio de posición, tamaño, rotación...) pero con la capacidad de especificar una duración para que el cambio se produzca de forma progresiva.

Para comprobar su funcionamiento usaremos el <div id="astronauta-t"> que contiene la imagen del astronauta. Pasa el cursor por encima de cada ejemplo para activar la transición correspondiente.

Para que transition funcione en todos los navegadores hay que repetir la propiedad anteponiendo los prefijos: -webkit- (Chrome y Safari), -moz- (Firefox), -o- (Opera), -ms- (Internet Explorer).

Transición con una única propiedad

Al pasar el cursor por encima del astronauta se producirá una transición de 1.5 segundos que desplazará el <div> 400 píxeles hacia la derecha.

#astronauta-t {
  transition: left 1.5s;
}

#astronauta-t:hover {
  position: relative;
  left: 400px;
}

En el primer bloque se define que la transición actuará sobre la propiedad left durante 1.5 segundos. El cambio de valor de left se coloca en :hover para que ocurra al pasar el cursor encima. Al usar left es obligatorio que position sea relative o absolute.

Transición con múltiples propiedades

Para animar varias propiedades a la vez con duraciones distintas se separan con ",". En este ejemplo, al pasar el cursor ocurren 6 cambios simultáneos:

  • Se desplaza 400px a la derecha (left) en 1.5s.
  • El fondo cambia a negro (background-color) en 2s.
  • La anchura crece a 225px (width) en 1s.
  • La altura crece a 200px (height) en 1s.
  • El padding-left se expande a 100px en 1s.
  • El padding-top se expande a 20px en 1s.
#astronauta-multi {
  background-color: #6a8f89;
  width: 100px;
  height: 100px;
  transition: left 1.5s, background-color 2s, width 1s, height 1s, padding-left 1s, padding-top 1s;
}

#astronauta-multi:hover {
  position: relative;
  left: 400px;
  background-color: black;
  width: 225px;
  height: 200px;
  padding-left: 100px;
  padding-top: 20px;
}

Los valores iniciales de background-color, width y height se definen en el primer bloque. Las propiedades que no se definan (como padding-left y padding-top) toman sus valores por defecto.

Las palabras mágicas

Aunque no es obligatorio, se puede especificar el tipo de movimiento y un retardo antes de que empiece la animación. La sintaxis completa es:

transition: [propiedad] [duración] [tipo de animación] [retardo];
transition: width 2s ease-out 0.5s;

Propiedades que se pueden animar

Entre las más habituales: all, background-color, border, border-radius, color, top, bottom, left, right, box-shadow, width, height, margin, opacity, padding, text-shadow, visibility, z-index, entre otras.

Tipos de animación (easing)

  • linear: velocidad uniforme en todo el recorrido.
  • ease: se acelera al inicio, se retarda un poco y vuelve a acelerar al final.
  • ease-in: empieza lento y va aumentando la velocidad progresivamente.
  • ease-out: empieza rápido y va desacelerando hasta el final.
  • ease-in-out: empieza y termina lento; la parte central es la más rápida.

Retardo

Tiempo en segundos que el navegador esperará antes de iniciar la animación (por ejemplo 0.5s).

En el ejemplo siguiente, al pasar el cursor, el astronauta esperará 0.5s y luego se volverá transparente durante 2 segundos con efecto ease-out (empieza rápido y se va frenando):

#astronauta-opacity {
  opacity: 1;
  transition: opacity 2s ease-out 0.5s;
}

#astronauta-opacity:hover {
  opacity: 0;
}

Compatibilidad con todos los navegadores

Para garantizar la compatibilidad con navegadores antiguos hay que repetir las propiedades con cada prefijo. El código del ejemplo anterior quedaría así:

#astronauta-opacity {
  opacity: 1;
  -webkit-opacity: 1;
  -moz-opacity: 1;
  -o-opacity: 1;
  -ms-opacity: 1;

  transition: opacity 2s ease-out 0.5s;
  -webkit-transition: opacity 2s ease-out 0.5s;
  -moz-transition: opacity 2s ease-out 0.5s;
  -o-transition: opacity 2s ease-out 0.5s;
  -ms-transition: opacity 2s ease-out 0.5s;
}

#astronauta-opacity:hover {
  opacity: 0;
  -webkit-opacity: 0;
  -moz-opacity: 0;
  -o-opacity: 0;
  -ms-opacity: 0;
}

La mayoría de navegadores modernos ya no necesitan prefijos para las transiciones, pero añadirlos garantiza compatibilidad con versiones antiguas.
