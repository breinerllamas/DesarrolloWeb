Columnas de texto

Con HTML5 y CSS3 se pueden crear textos en columnas de una manera muy sencilla, pudiendo configurar tanto el espacio entre columnas como las líneas de separación. Las tres propiedades clave son:

column-count

Define el número de columnas en las que se dividirá el texto.

column-count: 2;

column-gap

Define el espacio mínimo en píxeles que existirá entre las columnas.

column-gap: 10px;

column-rule

Define la línea separadora entre columnas. Acepta tres valores en orden: grosor, tipo de línea y color.

column-rule: 4px dotted gray;

Ejemplo práctico paso a paso

Paso 1 — Escribimos el contenido dentro de un <div>

Todo el texto o lista que queramos mostrar en columnas debe estar dentro de un <div> con un id para poder referenciarlo desde CSS. En este ejemplo usamos id="columnas" con una lista de reyes godos.

Se eliminan las etiquetas <ul> y </ul> para que el primer elemento no quede vacío y las columnas no se descuadren.

<div id="columnas">
  <li>Ataúlfo</li>
  <li>Sigerico</li>
  <li>Walia</li>
  <li>...</li>
</div>

Paso 2 — Añadimos el CSS

En el archivo CSS hacemos referencia al id del <div> e indicamos el número de columnas, la separación y el tipo de separador:

#columnas {
  column-count: 4;
  column-gap: 20px;
  column-rule: 4px dotted gray;
}

    • Tipos de líneas

    Los tipos de línea se usan tanto en column-rule como en la propiedad border. Estos son todos los valores disponibles:

    • none — sin línea separadora.
    • solid — línea sólida continua.
    • dotted — línea punteada.
    • dashed — línea discontinua (segmentos cortos).
    • double — doble línea paralela.
    • groove — línea en relieve 3D hacia adentro.
    • ridge — línea en relieve 3D hacia afuera.
    • inset — línea con color en bajorrelieve.
    • outset — línea que simula el relieve.

    • Compatibilidad con todos los navegadores

    Como los navegadores aún no comparten una sintaxis única, es necesario repetir las propiedades anteponiendo los prefijos de cada fabricante:

    • -webkit- para Chrome y Safari.
    • -moz- para Firefox.
    • -o- para Opera.
    • -ms- para versiones modernas de Internet Explorer.

    El código completo y compatible quedaría así:

    #columnas {
      column-count: 4;
      column-gap: 20px;
      column-rule: 4px double black;
      list-style: none;

      -webkit-column-count: 4;
      -webkit-column-gap: 20px;
      -webkit-column-rule: 4px double black;

      -moz-column-count: 4;
      -moz-column-gap: 20px;
      -moz-column-rule: 4px double black;

      -o-column-count: 4;
      -o-column-gap: 20px;
      -o-column-rule: 4px double black;

      -ms-column-count: 4;
      -ms-column-gap: 20px;
      -ms-column-rule: 4px double black;
    }
