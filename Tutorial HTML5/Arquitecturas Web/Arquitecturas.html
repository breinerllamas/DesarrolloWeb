Frontend - La Capa Visual

El frontend es todo lo que el usuario ve e interactúa en una aplicación web. Es la parte que se ejecuta en el navegador (browser) del cliente.

Tecnologías principales del Frontend

HTML

Estructura y contenido

HyperText Markup Language

CSS

Estilos y diseño

Cascading Style Sheets

JavaScript

Interactividad y lógica

Lenguaje de programación

Frameworks Frontend populares

  • React - Desarrollado por Facebook, basado en componentes
  • Vue.js - Progresivo y fácil de aprender
  • Angular - Framework completo de Google
  • Bootstrap - Framework CSS para diseño responsivo

Backend - El Motor Oculto

El backend es la parte que no ve el usuario, pero que hace funcionar la aplicación: servidores, bases de datos, APIs y lógica de negocio.

Bases de datos comunes

  • SQL - MySQL, PostgreSQL, SQL Server (relacionales)
  • NoSQL - MongoDB, Firebase, Redis (no relacionales)

Conceptos Fundamentales

¿Qué es una Aplicación Web?

Software accesible mediante navegador web, almacenado en servidores remotos. Ejemplos: Gmail, Facebook, Google Docs.

¿Qué es un Servidor Web?

Computadora o programa que entrega contenido web (HTML, CSS, JS) a los navegadores que lo solicitan.

API (Application Programming Interface)

Conjunto de reglas que permite comunicación entre diferentes aplicaciones. Las APIs REST son las más comunes en web.

Protocolo HTTP/HTTPS

Protocolo de transferencia de hipertexto. HTTPS es su versión segura con cifrado SSL/TLS.

Componentes Bootstrap

Sistema Grid

Bootstrap usa un sistema de 12 columnas responsive con clases como .col-md-4, .row.

Componentes UI

Botones, modales, tooltips, dropdowns, alertas y más componentes pre-diseñados.

Responsive Design

Clases como .img-fluid, .container-fluid para adaptarse a cualquier dispositivo.
