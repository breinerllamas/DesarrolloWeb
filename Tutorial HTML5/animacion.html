Animation

De una manera parecida a las transiciones, pero con una mayor potencialidad, en HTML5 tenemos a nuestra disposición las animaciones (animation).

Para empezar con algo sencillo, primero vamos a hacer paso a paso que el astronauta gire al colocar el cursor encima.

Sitúa el cursor encima del astronauta para ver la animación.

Paso a paso

1. Creamos el objeto a animar

Primero, creamos dentro de un <div> el objeto, imagen o texto que realizará la animación. En este caso usaremos el <div id="astronauta-anim-simple">, que tiene la imagen del astronauta definida como fondo en el CSS.

<div id="astronauta-anim-simple"></div>

2. Establecemos la animación

Seguidamente, en el código CSS, indicamos en el selector del <div> qué animación tendrá que reproducir, con un nombre y la duración en segundos.

#astronauta-anim-simple:hover {
  animation: animacion_astronauta 2s;
}

El nombre de la animación (animacion_astronauta) y la duración (2s) son valores obligatorios, pero existen valores opcionales que nos permiten personalizarla:

  • animation-timing-function: Define el tipo de aceleración (easing).
    • linear: la velocidad es siempre la misma durante toda la animación.
    • ease: la animación comienza lenta, después rápida y acaba lento.
    • ease-in: la animación empieza lenta y va acelerándose progresivamente.
    • ease-out: la animación empieza rápida y va decelerando progresivamente.
    • ease-in-out: la animación empieza y acaba lenta; la parte central va rápida.
  • animation-iteration-count: Número de veces que se repetirá la animación.
    • infinite: la animación se repite indefinidamente.
    • número de repeticiones: se repite el número de veces indicado (por defecto es 1).
  • animation-direction: Establece la dirección de la animación.
    • normal: la animación se reproduce siempre tal y como la hemos definido.
    • alternate: primero se reproduce en la dirección definida y luego al revés (y así sucesivamente).
  • animation-delay: Tiempo de espera antes de que se inicie la animación.
    • El número de segundos seguido de s (por ejemplo, 1s).
  • animation-fill-mode: Indica cómo quedará el elemento cuando la animación finalice.
    • none: al acabar se aplicarán los estilos CSS normales del elemento.
    • forwards: el elemento se quedará tal y como acabó la animación (100%).
    • backwards: el elemento se quedará con las propiedades del inicio (0%).

El ejemplo anterior podría complementarse con estas propiedades opcionales:

#astronauta-anim-simple:hover {
  animation: animacion_astronauta 2s;
  animation-timing-function: ease-out;
  animation-iteration-count: infinite;
  animation-direction: alternate;
  animation-delay: 1s;
}

3. Definimos cómo será la animación

Dentro del mismo código CSS, definimos en qué consistirá la animación usando @keyframes con el nombre que asignamos antes.

@keyframes animacion_astronauta {
  0% {
    transform: rotate(0deg);
  }
  50% {
    transform: rotate(90deg);
  }
  100% {
    transform: rotate(0deg);
  }
}

Toda la animación se puede dividir en tantos pasos como sea necesario, aunque lo mínimo son 2: inicio (0%) y final (100%).

En este ejemplo hay 3 pasos. Al inicio (0%) el astronauta tiene una rotación de 0 grados; en el 50% habrá rotado 90 grados hacia la derecha; y al 100% volverá a su posición original.

Ejemplo práctico complejo

El siguiente ejemplo es más complejo: el astronauta no solo rota, sino que también se desplaza hacia abajo durante la animación, y al llegar al punto máximo de rotación permanece un momento antes de volver a su posición inicial.

Se usa el parámetro infinite para que la animación sea cíclica.

Para que el astronauta también baje al girar, añadimos la propiedad top en cada paso de la animación:

@keyframes animacion_astronauta_complejo {
  0% {
    transform: rotate(0deg);
    top: 0px;
  }
  50% {
    transform: rotate(90deg);
    top: 57px;
  }
  100% {
    transform: rotate(0deg);
    top: 0px;
  }
}

Siempre que usemos top, left o right, debemos indicar si la posición es relativa o absoluta, añadiendo position: relative en el selector del div.

Para que el astronauta permanezca un momento en la posición girada, añadimos un paso extra y ajustamos los porcentajes centrales:

@keyframes animacion_astronauta_complejo {
  0% {
    transform: rotate(0deg);
    top: 0px;
  }
  42% {
    transform: rotate(90deg);
    top: 57px;
  }
  60% {
    transform: rotate(90deg);
    top: 57px;
  }
  100% {
    transform: rotate(0deg);
    top: 0px;
  }
}

Al tratarse de una animación de 2 segundos:

  • El astronauta baja y gira en 0.84 segundos (del 0% al 42%).
  • Permanece rotado durante 0.36 segundos (del 42% al 60%).
  • Vuelve a su posición inicial en 0.80 segundos (del 60% al 100%).

Compatibilidad con todos los navegadores

Como ocurre con otras funcionalidades de HTML5, es necesario añadir los prefijos para que todos los navegadores entiendan estos códigos. En este caso el proceso es todavía más extenso, ya que hay que repetir todo el bloque @keyframes con cada prefijo:

@keyframes animacion_astronauta_complejo { ... }

@-webkit-keyframes animacion_astronauta_complejo {
  0% { -webkit-transform: rotate(0deg); top: 0px; }
  42% { -webkit-transform: rotate(90deg); top: 57px; }
  60% { -webkit-transform: rotate(90deg); top: 57px; }
  100% { -webkit-transform: rotate(0deg); top: 0px; }
}

@-moz-keyframes animacion_astronauta_complejo {
  0% { -moz-transform: rotate(0deg); top: 0px; }
  42% { -moz-transform: rotate(90deg); top: 57px; }
  60% { -moz-transform: rotate(90deg); top: 57px; }
  100% { -moz-transform: rotate(0deg); top: 0px; }
}

@-o-keyframes animacion_astronauta_complejo {
  0% { -o-transform: rotate(0deg); top: 0px; }
  42% { -o-transform: rotate(90deg); top: 57px; }
  60% { -o-transform: rotate(90deg); top: 57px; }
  100% { -o-transform: rotate(0deg); top: 0px; }
}

Y la especificación de la animación en el selector del div, también con todos los prefijos:

#astronauta-anim-complejo {
  animation: animacion_astronauta_complejo 2s;
  -webkit-animation: animacion_astronauta_complejo 2s;
  -moz-animation: animacion_astronauta_complejo 2s;
  -o-animation: animacion_astronauta_complejo 2s;
}

En este código todavía no se ha añadido el prefijo -ms- para Internet Explorer, ya que este navegador no soporta animation.
